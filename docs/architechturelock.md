# Architecture Lock

This repository follows a fixed architecture:

- frontend → React + Vite + Tailwind (PWA)
- backend → Node.js + Fastify (real-time APIs)
- streaming → FFmpeg + Nginx (HLS/DASH)
- ml → Python (training + serving)
- infra → Docker Compose (local orchestration)

No folders or services will be added
without updating this document.
